English Summary

Architecture

  • frontend/: React + Vite + TypeScript dashboard.
  • proxyapi_core/: Go proxy + management API backend.
  • Default endpoints: backend http://localhost:8317, frontend http://localhost:5173.

Requirements

  • Go >= 1.26.0
  • Node.js >= 18 (20+ recommended)
  • npm >= 9
  • Docker + Docker Compose (optional)

Quick Start

# Windows quick run
run-dev.bat

# backend only
run-real.bat

Manual Setup

cd proxyapi_core
cp config.example.yaml config.yaml
# set api-keys and remote-management.secret-key
go run ./cmd/server

cd ../frontend
npm install
npm run dev

Management endpoints require remote-management.secret-key. If empty, /v0/management/* returns 404.

Use the Proxy in SDK/CLI

curl http://localhost:8317/v1/models \
  -H "Authorization: Bearer my-personal-key"

Set your tool base URL to http://localhost:8317/v1, and use a key from api-keys.

Troubleshooting nhanh

Vấn đề Cách xử lý
401 UnauthorizedKiểm tra key trong api-keys của config.yaml
404 endpoint managementĐặt remote-management.secret-key
Dashboard usage trốngBật usage-statistics-enabled: true
Log viewer trốngBật logging-to-file: true
Port bị chiếmĐổi port: trong config.yaml

Ghi chú bảo mật

  • Không commit file secrets thật trong proxyapi_core/config.yaml.
  • Không commit dữ liệu runtime: logs/, auths/, cache/temp.
  • Nên xoay vòng API key và secret định kỳ.
